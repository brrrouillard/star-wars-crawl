# Star Wars Crawl
## Getting Started

### Prerequisites
Any web browser.

### Installing and Running
Launch [index.html](https://brrrouillard.github.io/star-wars-crawl/).

## Purpose
I built this webpage during my internship at BeCode. The goal was to remake the famous opening crawl from Star Wars.
